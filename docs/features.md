# 功能改动总览

本文档按功能域记录当前所有玩家可见改动。具体物品数值和配方见 `design/item.md`，技术原因见 `design/`。

## 经济系统

- 新增四档实体撒币。
- 实体撒币可以在相邻面额间兑换。
- 新增玩家个人撒币账户。
- 玩家可以把实体撒币存入账户。
- 玩家可以从账户提现实体撒币。
- 账户入口显示在玩家背包界面。

## 新增物品与方块

- 新增小撒币、中撒币、大撒币、巨撒币。
- 新增高级红石核心。
- 新增撒币机方块。
- 高级红石核心用于合成撒币机。
- 撒币机需要较高级工具挖掘。

## 撒币机交易

- 新增撒币机方块。
- 撒币机支持典当配置允许的物品。
- 典当收益直接进入玩家账户。
- 撒币机支持赎回全局库存中的物品。
- 赎回消耗账户余额。
- 当某物品全局库存为 0 时，撒币机支持购买该物品。
- 购买价格为 `4 * pawn_price`。

## 撒币机库存

- 撒币机库存是服务器全局库存。
- 任意一台撒币机、任意玩家看到的都是同一份库存。
- 打破撒币机不会掉出全局库存。
- 库存保存真实 `ItemStack`，会保留组件差异。

## 撒币机界面

- 主界面显示可典当物品网格。
- 主界面支持搜索。
- 主界面支持快速典当槽。
- 主界面 `Shift+Click` 背包物品会放入快速典当槽；槽内已有物品时不会合并。
- 点击物品进入详情页。
- 详情页显示库存、典当收益、赎回或购买价格、账户余额。
- 赎回和购买数量输入框初始为空。
- 当花费超过账户余额时，花费数字显示为红色。

## 潜影盒典当

- 主界面快速典当槽中的非空潜影盒会典当盒内可典当物品。
- 非空潜影盒典当后，盒子本身留在槽内。
- 不可典当的盒内物品会留在盒内。
- 空潜影盒需要确认后才会典当盒子本身。

## 数据安全

- 典当槽中的物品会镜像保存到玩家数据。
- 正常关闭撒币机界面时，典当槽物品会回到玩家背包。
- 退出服务器时，典当槽物品会回到玩家背包。
- 重新进入服务器时，如果存在挂起典当槽物品，会自动归还。
