# 撒币机 UI

相关文件：

- `client/SabiPawnMachineScreen.java`
- `SabiPawnMachineMenu.java`

## 页面模式

- `GRID`：主列表页。
- `DETAIL`：选中物品后的交易页。
- `PAWN_CONFIRM`：主页面快速典当确认页。
- `CONTAINER_CONTENTS_CONFIRM`：主页面非空容器内容典当确认页。
- `EMPTY_CONTAINER_CONFIRM`：主页面空容器典当确认页。
- `NOTICE`：主页面操作结果提示页，需要玩家点击确认后返回主列表页。

## 共用弹窗布局

`PAWN_CONFIRM`、`CONTAINER_CONTENTS_CONFIRM`、`EMPTY_CONTAINER_CONFIRM` 和 `NOTICE` 使用 `SabiPawnMachineScreen` 中的共享弹窗布局 helper 统一管理布局。

- 弹窗内容框、标题、物品图标、物品名称、正文起点和正文右边界都由代码中的布局常量决定。
- 普通弹窗正文从物品名称下方开始；提示弹窗正文从弹窗正文区域左侧开始。
- 弹窗正文作为单段文本自动换行。
- 双按钮弹窗使用确认/取消按钮；单按钮提示弹窗只显示确认按钮。

## 主列表页

关键控件：

- 搜索框
- 快速典当槽：menu slot 0
- 快速典当按钮
- 物品表格：固定行列的可滚动网格，具体坐标和尺寸以 `SabiPawnMachineScreen` 为准。

行为：

- 支持鼠标滚轮滚动。
- 支持搜索中文显示名和完整物品 ID。
- `Shift+Click` 玩家背包物品时，只会在快速典当槽为空时放入该槽。
- 普通物品典当确认页、非空容器内容典当确认页和空容器典当确认页使用同一套弹窗位置和确认/取消按钮位置。
- 如果非空容器典当后仍有不可典当物品留在容器内，界面会显示提示弹窗；该弹窗只有确认按钮，正文作为单段文本按灰色内容框宽度自动换行。

排序：

1. 全局库存数量降序。
2. 配置文件原始顺序升序。

药水、喷溅型药水、滞留型药水和药箭在列表中显示基础物品名。
这些物品当前没有按具体药水效果拆分，使用空 `ItemStack` 的悬浮名会显示成不可合成/默认变体，容易误导玩家。

## 详情页

展示内容：

- 物品图标
- 名称
- 全局库存数量
- 典当收益
- 赎回花费或购买花费
- 账户余额或花费/余额

详情页典当槽要求放入当前选中的同一种物品。

## 输入处理

屏幕覆写 `keyPressed`。

- 如果搜索框或数量框聚焦，按 E 不关闭界面。
- 如果编辑框未聚焦，按 E 按原版逻辑关闭。
