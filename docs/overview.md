# Mod 概览

Sabi 是一个基于 Fabric 的 Minecraft Java 版 Mod。

- Mod ID：`sabi`
- Java 包名：`top.sabi`
- Minecraft：`26.2`
- Fabric Loader：`0.18.4`
- Fabric API：`0.154.2+26.2`
- Java/JDK：`25`
- 当前 Mod 版本：`1.2.0`

## 当前主题

当前核心是撒币经济系统。

它包括：

- 四档实体货币
- 玩家个人撒币账户
- 背包界面账户入口
- 撒币机典当、赎回和购买
- 全服务器共享的撒币机库存
- 可人工编辑和自动推导的撒币机价格规则

## 已实现

- 小撒币、中撒币、大撒币、巨撒币。
- 绿宝石合成中撒币。
- 玩家个人账户。
- 账户提现和全部存入。
- 撒币机方块和合成配方。
- 撒币机全局库存。
- 撒币机典当收益入账户。
- 撒币机赎回和购买扣账户余额。
- 撒币机容器物品快速典当。
- 价格规则生成和交互式编辑脚本。

## 尚未实现

- 德州扑克玩法。
- 撒币机按物品组件、耐久、附魔细分价格。
- 撒币机库存浏览中选择同一种物品的不同组件栈。
