## 撒币账户

- 账户不依赖钱包物品，而是直接绑定到玩家
- HUD 会在盔甲显示区域附近显示一个撒币账户方框
- 点击方框会打开撒币账户界面
- 界面功能：
  - 输入指定数量并点击“提现”，会按巨撒币、大撒币、中撒币、小撒币的顺序发放实体货币
  - 点击“全部存入”，会把背包和装备槽中的所有实体撒币兑换为账户余额
- 相邻面值汇率：`64`

## 高级红石核心

- 物品 ID：`sabi:advanced_redstone_core`
- 显示名称：高级红石核心
- 创造模式位置：材料栏
- 获取方式：红石块、石英、红石比较器、回响碎片和下界合金锭合成
- 用途：撒币机的核心合成原料
- 贴图：红石核心外框，中间嵌有回响碎片和合金质感核心

## 撒币机

- 方块 ID：`sabi:sabi_machine`
- 显示名称：撒币机
- 创造模式位置：材料栏
- 硬度：与黑曜石同级，硬度 `50.0`
- 爆炸抗性：`1200.0`
- 挖掘要求：需要钻石级镐类工具
- 获取方式：黑曜石、绿宝石块、末影箱和高级红石核心合成
- 当前功能：右键打开撒币机界面，可从配置允许的物品列表中选择一种物品进行典当或赎回
- 默认规则：`1` 个物品典当为 `1` 个小撒币实体；`2` 个小撒币实体可赎回 `1` 个已经进入撒币机的物品
- 配置文件：`src/main/resources/data/sabi/sabi_machine/items.json`
  - 默认从当前版本的物品注册表生成候选列表
  - 通过 `exclude_items` 和 `exclude_id_suffixes` 排除非生存/不应典当的物品
  - 通过 `default_pawn_price`、`default_redeem_price` 和 `price_overrides` 调整价格
- 贴图：黑曜石机壳、绿宝石正面面板、红石指示灯和顶部核心槽

## 注意事项

Minecraft 26.1.2 的普通容器和槽位仍有堆叠数量限制，所以实体撒币统一使用 `64` 堆叠上限。大额余额由玩家账户系统记录，实体货币主要用于交易、掉落和展示。
