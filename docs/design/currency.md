# 实体撒币与兑换

相关文件：

- `CurrencyDenomination.java`
- `CurrencyItem.java`
- `src/main/resources/data/sabi/recipe/*.json`

## 面额

具体物品 ID、显示名和账户价值见 `item.md`。

设计规则：

- 实体货币只负责展示、交易和掉落。
- 大额长期余额放在个人账户里。
- 撒币本身不可被撒币机典当。

## 兑换实现

当前只提供绿宝石到中撒币这一条普通无序配方，用于把原版绿宝石按价格配置中的等价关系转换成实体撒币。

实体撒币之间不提供合成互换。玩家如果需要长期存储或大额换算，应使用个人账户。
